2021-11-29

今天部署博客到github, 但是呢, 一直部署不上去, 先是提示什么SSL错误, 然后我找到网上一个教程;
前面的都做完了, 就差最后一步:设置repository没执行.
我想: 我本地用的repo, 如下:

deploy:
    type: git
    repo: https://github.com/womine/womine.github.io
    branch: master
    name: womine

而且, hexo-deployer-git 中的示例也是用的repo, 应该可以噻!
然后我就不执行了, 结果还是出错, 试了又试, 始终不行, 迫于无奈, 还是改成了repository, 最后_config.yml部分配置如下:

deploy:
  type: git
  # repo: https://github.com/womine/womine.github.io
  repository: git@github.com:womine/womine.github.io.git
  branch: master
  name: womine
  email: XXXXXXXXXXXXXXXXXXXXXXXX@XX.XXX
  token: demo_demo_demo

然后执行 hexo d; ok, 成功了 ┑(￣Д ￣)┍.

既然可以, 就先这样吧, 以后再查查为啥子了~
