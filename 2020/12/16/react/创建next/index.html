创建next.js

创建一个 next.js 项目

初始化一个 node 项目

命令:

npm init

初始化以后会有一个 package.json
在 package.josn 中配置:

{
  "name": "womine.index",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
    "dev": "next",
    "build": "next build",
    "start": "next start -p 8080"
  },
  "author": "womine",
  "license": "ISC",
  "dependencies": {
    "@zeit/next-css": "^1.0.1",
    "antd": "^4.8.0",
    "babel-plugin-import": "^1.13.1",
    "moment": "^2.29.1",
    "next": "^9.5.5",
    "react": "^17.0.1",
    "react-dom": "^17.0.1"
  }
}

根目录创建文件夹 pages, components, public
在 pages 中创建文件 index.ts
这个 index.ts 就是我们项目的主页了

然后执行命令;

npm run build

build结束后, 执行

npm run start

然后访问: localhost:8080
当然, 也可以用命令

npm run dev

然后访问 localhost:3000 访问开发版本, 开发版本可以动态更新, 适合开发的时候用.

react next.js
react next.js
