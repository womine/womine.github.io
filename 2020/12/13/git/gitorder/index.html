git常用命令

Git常用命令

删除远程分支(分支名称: darft)

git push origin --delete darft

增加文件到办本地:

git add [文件名]

提交到本地:

git commit -m "备注" (备注必须存在, 随便写点都可以)

提交到远程:

git push

从远程仓库获取最新的版本到本地的 tmp 分支上

git fetch

合并分支

git merge
